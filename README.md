# Atividades Técnicas de Desenvolvimento de Algoritmos

Este repositório serve como um portfólio para apresentar as atividades práticas e teóricas desenvolvidas para a disciplina de **Técnicas de Desenvolvimento de Algoritmos**.

---

## Conteúdo

Cada pasta neste repositório representa um conjunto de exercícios ou um projeto específico, demonstrando a aplicação de diferentes conceitos, como:

* **Estruturas de Dados:** Implementações de listas, pilhas, filas e outras estruturas.
* **Algoritmos:** Resolução de problemas utilizando técnicas de busca, ordenação e manipulação de dados.
* **Análise de Complexidade:** Estudo e aplicação da notação `Big O` para avaliar a eficiência de soluções.

---

## Estrutura do Repositório

* Contém os arquivos de código-fonte e, se necessário, documentação adicional para uma atividade específica.

Sinta-se à vontade para explorar e analisar as soluções.
